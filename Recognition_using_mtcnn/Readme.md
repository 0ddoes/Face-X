## Ouick Start
- Clone this repository.
`git clone https://github.com/akshitagupta15june/Face-X.git`
- Change Directory.
`cd Recognition_using_mtcnn`
- Run File.
`python mtcnn.py`
## Dependencies
- `pip install tensorflow`
- `pip install mtcnn`
- `pip install matplotlib`
